O que é HTML?

HTML significa Hypertext markup language, que é basicamente a linguagem que o browser lê, compreende e nos apresenta de uma forma mais amigável. Apesar de conter "language" no nome, o HTML não é propriamente uma linguagem de programação. HTML é o padrão de linguagem de marcação de texto para a estruturação de páginas web. Ele consiste em um série de elementos e TAGs que diz ao browser como mostrar e estruturar o seu conteúdo. 1

O que são TAGs?

TAGs são todos os elementos que um tipo de característica do texto que será interpretada pelo browser. O navegador lê todo o HTML e suas tags, para então apresentar ao client como se deve. Ele não irá mostrar as TAGs em si, mas sim as suas características junto do texto (e outros elementos que serão estudados mais pra frente, como CSS, Javascript e todo o universo que se pode criar com esse conjunto). 1
